# Toolbox 项目总说明

项目已从自动化测试平台收缩为轻量效率工具箱。

保留范围：

- 效率工具：数据生成、文件快传、图片工具、JSON 工具、转码工具。
- 系统配置：集成配置、用户管理。
- 基础能力：登录认证、健康检查、MySQL、Nginx、Docker Compose。

已移除范围：

- 项目、测试对象、接口/UI 用例、测试任务、Mock、执行记录、结果中心、问题定位、质量分析和测试报告。
- Worker、Redis、Playwright、pytest 执行调度和 JMeter 辅助代码。

历史业务表不会在启动时主动删除，但当前应用不再映射或访问这些表。如需清理数据库，请另行备份后执行数据库迁移，不要直接删除生产数据。
