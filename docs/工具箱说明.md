## 数据生成

接口：`POST /api/v1/tools/data-generator`

支持中国大陆手机号格式、Twilio 模拟号码、服务端受控接收号码和校验位正确的合成身份证号码。随机手机号不得用于拨打或发送短信。

受控短信号码通过 `.env` 配置：

```env
TEST_SMS_PHONE_NUMBERS=+14155550100,+442079460018
```

## 部署

```bash
cd /opt/automation-platform
bash deploy/scripts/deploy.sh
```

部署后检查：

```bash
docker compose ps
curl http://127.0.0.1/api/health
```
