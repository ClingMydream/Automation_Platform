- 代码和文档一起提交，避免“代码变了，说明没跟上”。

## 2026-07-09 补充：UI 自动化执行录屏

本次完成：

- HTML 测试报告导出支持带出 UI 执行录屏。
- 新增录屏大小保护，默认超过 `25MB` 不内嵌到 report，避免数据库 JSON 字段被大文件撑爆。

影响范围：

- frontend: `frontend/src/modules/90-live-run/`、`frontend/src/modules/08-run-history/`、`frontend/src/shared/reportExport.js`、`frontend/src/styles/app.css`

- deploy: 无 docker-compose 结构变更

配置说明：

- `UI_RECORD_VIDEO=true`：默认开启 UI 录屏。
- `UI_RECORD_VIDEO=false`：关闭 UI 录屏。
- `UI_RECORD_VIDEO_MAX_MB=25`：单个录屏内嵌到 report 的最大 MB 数。

## 2026-07-09 补充：性能测试最小执行闭环

本次完成：

- 性能任务支持读取 `config.performance_scenario_ids`。
- 启动性能任务时创建执行批次，并为每个性能场景创建 `case_type=performance` 的执行记录。
